# Final Refinement Prompt

Use this prompt for the next brand pass focused only on the chosen direction.

## Prompt

Create a final refinement board for the brand of a desktop PDF application named PDF-StitchUI. Focus only on the chosen direction: Tile Sequence, Balanced variant. The identity should show 3 to 4 ordered document tiles or page cards resolving into one final assembled document, clearly expressing sequencing and stitching into a finished PDF. The style must feel like a serious native desktop productivity tool: crisp geometry, strong silhouette, clean spacing, and excellent small-size legibility. Do not imitate Adobe branding.

Show the following on a clean light presentation board:

- primary app icon
- simplified small-size icon
- horizontal wordmark reading PDF-StitchUI
- square social or repo avatar mark
- compact installer or release badge variant
- a small color palette strip

Visual direction:

- balanced, polished, and professional
- more distinctive than a generic PDF icon
- less busy than a motion-heavy concept
- deep document red as the anchor color
- charcoal and steel gray for structure
- optional restrained muted blue accent only if it improves hierarchy

Composition requirements:

- make the tile-to-final-document progression obvious
- keep the final icon viable at small sizes
- emphasize Stitch as the memorable idea within the overall mark
- avoid decorative sewing cues that feel crafty rather than technical
- avoid gradients or effects that reduce clarity at icon scale

The result should look like a near-final software brand system, not a loose brainstorming board.
