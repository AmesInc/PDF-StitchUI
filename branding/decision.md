# Branding Decision

## Selected Direction

- Primary concept: `Tile Sequence`
- Selected variant: `Balanced`

## Why This Was Chosen

- It matches the actual product behavior better than a generic PDF mark.
- It communicates ordered assembly, which is the core stitch workflow.
- It stays legible at app-icon size without becoming visually noisy.
- It gives the product a stronger identity than a plain document icon.
- It can scale into a broader icon family for installers, releases, and toolbar graphics.

## Intended Brand Meaning

The logo should suggest that multiple ordered document tiles resolve into one final assembled PDF.
The emphasis is on sequencing, control, and clean assembly rather than abstract file management.

## Practical Usage Goals

- Primary app icon
- Simplified small-size icon
- Horizontal wordmark
- Square repo/social mark
- Installer and release badge variant
